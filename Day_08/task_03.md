```python
list.append(element)
```
Append refers to adding an element at the very end of the list , only a slight adjustment of a single block.Time Complexity O(1)

```python
list.insert(element,position)
```

Inserting at a position of the list needs shifting of elements. It goes by time complexity of O(N)
